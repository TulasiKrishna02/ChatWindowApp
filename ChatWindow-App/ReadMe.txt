To run the Application Following these Steps:
==============================================
1.Navigate to the project directory: cd chatroom-backend

2.Set up the backend server:

3.Install the necessary dependencies: mvn clean install

4.Start the Spring Boot server: mvn spring-boot:run Set up the frontend:

5.Install the necessary dependencies: cd chatroom-ui && npm install

6.Start the React development server: npm run dev

7.Open your web browser and visit http://localhost:5173 to access the chat application.
